# image-watermark-tool

A pure JavaScript image watermark application built using HTML5 Canvas.

## Features
- Base image upload
- Watermark upload (PNG supported)
- Resize, opacity, rotation controls
- 9 positioning options
- Live preview
- Download final image

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- HTML5 Canvas

Runs locally without setup.
